## 🚀 Deployment on GitHub (Quick Start)
1. Push `index.html`, `style.css`, `script.js`, and this `README.md` into a new, public GitHub repository.
2. Head into repository **Settings** -> **Pages**.
3. Under *Build and deployment*, assign the build source to the `main` branch. 
4. Click **Save** and wait for the live web link!
